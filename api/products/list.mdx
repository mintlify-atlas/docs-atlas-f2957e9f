---
title: List Products
description: Retrieve a list of all products with optional pagination
---

## Endpoint

```
GET /api/v1/products
```

Retrieves all products from the system.

## Pagination Endpoint

```
GET /api/v1/products/pagination
```

Retrieves products with pagination support.

### Query Parameters

<ParamField query="page" type="integer" default="0">
  The page number to retrieve (zero-based index). First page is 0.
</ParamField>

<ParamField query="pageSize" type="integer" default="10">
  The number of products to return per page.
</ParamField>

## Response

<ResponseField name="products" type="array">
  Array of product objects
  
  <Expandable title="Product Object">
    <ResponseField name="id" type="integer">
      Unique identifier for the product
    </ResponseField>
    
    <ResponseField name="name" type="string">
      The name of the product
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Detailed description of the product
    </ResponseField>
    
    <ResponseField name="sku" type="string">
      Stock Keeping Unit identifier
    </ResponseField>
    
    <ResponseField name="category_id" type="integer">
      ID of the product's category
    </ResponseField>
    
    <ResponseField name="price" type="number">
      Selling price of the product
    </ResponseField>
    
    <ResponseField name="cost_price" type="number">
      Cost price of the product
    </ResponseField>
    
    <ResponseField name="weight_kg" type="number">
      Weight in kilograms
    </ResponseField>
    
    <ResponseField name="is_active" type="boolean">
      Whether the product is active
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Creation timestamp (ISO 8601 format)
    </ResponseField>
    
    <ResponseField name="product_dimension_id" type="integer">
      Associated dimensions record ID
    </ResponseField>
    
    <ResponseField name="product_inventory_id" type="integer">
      Associated inventory record ID
    </ResponseField>
    
    <ResponseField name="image_ids" type="array">
      List of associated image IDs
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

### List All Products

```bash
curl -X GET "https://api.furniture.com/api/v1/products" \
  -H "Accept: application/json"
```

### List Products with Pagination

```bash
curl -X GET "https://api.furniture.com/api/v1/products/pagination?page=0&pageSize=20" \
  -H "Accept: application/json"
```
