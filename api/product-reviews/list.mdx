description: 'Retrieve a list of all product reviews'
---

## Endpoint

```
GET /api/v1/product-reviews
```

Retrieve a list of all product reviews in the system.

## Response

Returns an array of product review objects.

<ResponseField name="reviews" type="array">
  Array of product review objects
  
  <ResponseField name="id" type="integer" required>
    Unique identifier for the review
  </ResponseField>
  
  <ResponseField name="productId" type="integer" required>
    ID of the product being reviewed
  </ResponseField>
  
  <ResponseField name="userId" type="integer" required>
    ID of the user who created the review
  </ResponseField>
  
  <ResponseField name="rating" type="integer" required>
    Rating given to the product (typically 1-5)
  </ResponseField>
  
  <ResponseField name="comment" type="string">
    Text comment or review description
  </ResponseField>
  
  <ResponseField name="createdAt" type="string">
    Timestamp when the review was created (ISO 8601 format)
  </ResponseField>
</ResponseField>

## Status Codes

<ResponseField name="200" type="OK">
  Successfully retrieved list of product reviews
</ResponseField>

## Example Request

```bash
curl -X GET "https://api.example.com/api/v1/product-reviews"
```

## Notes

- This endpoint returns all reviews in the system without filtering
- For paginated results, use the [pagination endpoint](/api/product-reviews/search#pagination)
- To get reviews for a specific product, use the [product reviews endpoint](/api/product-reviews/search#by-product)
- Results are not ordered by default; use the ordered endpoint for chronological sorting
